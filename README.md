# Webpack

Webpack project is a basic project that shows how to setup webpack to be used as your project bundler in a development mode


## Built With

- Git
- Github
- HTML
- CSS
- Javascript
- webpack

## Getting Started

$ git clone git@github.com:rugwizangoga/Webpack

$ cd Webpack

$ run npm start
